# План задач: Mafbase Stream

Декомпозиция фичи из [BRD](mafbase_stream_brd.md) на последовательные задачи. Каждая задача — отдельный testable инкремент: после её завершения можно проверить результат руками или автотестом.

Где результат отдельной задачи нельзя продемонстрировать (например, чистый C++ модуль без UI), задача объединена со следующей, чтобы получить наблюдаемый эффект.

## Ключевые точки демонстрации

- После **03** — видно «железную» работоспособность RTMP-клиента (без UI).
- После **06 / 09** — рабочий мобильный RTMP-стрим без оверлея на каждой платформе.
- После **11** — функционально завершённый pipeline (стрим + оверлей + синхронизация).
- После **18** — релиз-кандидат.

## Допущения

Этот план опирается на допущения BRD (раздел 1.3, 4.7, 5.3). Перед началом работ — пройтись по «ДОПУЩЕНИЕ»-меткам в BRD и подтвердить.
